## 📌 Project Overview
An autonomous robot that detects and extinguishes fire while avoiding
obstacles, built on an Arduino UNO with a 3-mode prioritized FSM
(Fire Mode → Obstacle Mode → Normal Mode).

**Total Cost:** ~BDT 3,800 (active components)

## ⚙️ Hardware Used
- Arduino UNO (ATmega328P)
- HC-SR04 Ultrasonic Sensor
- IR Flame Sensors ×2
- L298N Motor Driver
- 4WD Chassis + DC Motors
- Submersible Water Pump + Relay
- 12V Li-Ion + 9V Battery (dual rail)
